
Description : Build a unsorted, unbalanced binary tree, serialize to disk and de-serialize from disk

Solution:
Two methods are implemented

Method1:
1.Build Tree
2.Serialize and Deserialze using Java Object Serialisation mechanism

Method 2:
1.Build Tree.
2.Use depth traversal to serialise the tree and de-serialize 

How to build and run with test:

mvn clean package
